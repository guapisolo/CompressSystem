以下操作均在 build 文件夹内部进行

先生成编译命令：

```
cmake ..
```
然后编译：
```
make 
```

会找到一个叫 Demo 的可执行文件，gdb Demo 或者 ./Demo 就能跑了

注意环境是linux，我用虚拟机跑的

如果修改以后跑不起，就跑下面这个全部重新编译：

```
rm * -rf && cmake .. && make
```
